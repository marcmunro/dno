NAME

dno_requote - Process the quotes in an Arduino recipe command into a format suitable for bash

SYNOPSIS

echo “COMMAND” | dno_requote

DESCRIPTION

dno_requote is a filter that takes Arduino recipe commands as stdin, and converts the quoting within these commands to be compatible with bash.

It is not clear to the author what the original quoting is compatible with.

This is part of the dno Arduino software builder package.
